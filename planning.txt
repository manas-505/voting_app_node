Voting application
what?
A functionality where user can give vote to the given set of candidates.


Models?(Scheme in database) 
Routes?(Endpoint Types )


Voting app functionality

1. User sign in/ sign up
2. See the list of candidates
3. vote one of the candidates ,after voting users cannot vote again
4. There is a route show the list of candidates and their live votes count sorted by their vote number
5. User data must contain their one unique goverment ID proof named :Aadhar card
6. There should be one admin Who can only maintain the tables of the candidates and he cannot able to vote at all
7. User can change their password
8. User Can login Only with aadhar card And password
9. Admin cannot  Vote at all


--------------------------------------------------------------------

Routes(Endpoints)

User authentication
    / Signup:POST - Create a new user account.
    / Login:POST- Log into one existing account.(Aadhar card number And Password )

Voting:
    / Candidates: GET- Get the list of candidates
    /Vote : Candidate ID:POST- Bought the specific Candidates


Vote Count:
    /vote/count :GET- Get the list of candidates sorted by their vote count

User profile:
    / Profile:GET- Get the user profiles Information
    / Profile/ Password:PUT - Change the user's Password

Admin candidate Management:
    /Candidate:POST- Create a new Candidate
    /Candidates/: Candidateid: PUT- Update unexisting candidate
    /Candidates/: Candidateid: DELETE- Delete a candidate from the list
